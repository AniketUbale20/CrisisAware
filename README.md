> **A dual-domain fusion platform that combines real-time emergency monitoring with AI-driven financial fraud detection to optimize civic safety and resource allocation.**

## ðŸ“Œ Overview
CrisisAware addresses the "siloed data" problem in smart cities. During natural disasters, financial fraud often spikes, but emergency teams and banks work in isolation. This platform aggregates live data from **USGS (Earthquakes)** and **NASA FIRMS (Wildfires)** and correlates it with simulated financial transaction logs to detect geo-temporal anomalies.
